# YOLOv11n 手部检测项目 - 虚拟环境设置指南

**如果需要使用pycharm进行虚拟环境管理，务必安装Anconda/miniconda，并python版本选择3.11**

## 1. 创建虚拟环境

```bash
# 创建虚拟环境
conda create -n YOLOv11n python=3.11

# 激活虚拟环境
conda activate YOLOv11n
```

## 2. 安装依赖

### Step 1 下载cuda

```
python -m pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### Step 2 下载其他依赖

```
pip install -r requirements.txt
```

## 3. 验证安装

在激活的虚拟环境中执行：

```bash
python
import torch
import ultralytics
print("环境设置成功！")
```

## 4.CUDA支持检查

```python
import torch
print(f"PyTorch版本: {torch.__version__}")
print(f"CUDA可用: {torch.cuda.is_available()}")
if torch.cuda.is_available():
    print(f"GPU设备: {torch.cuda.get_device_name(0)}")
```

## 注意事项

1. **确保Python版本**：项目需要Python 3.11+
2. **激活虚拟环境**：在命令行中操作时需激活虚拟环境
3. **退出虚拟环境**：使用 `deactivate` 命令退出虚拟环境
